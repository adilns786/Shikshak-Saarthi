```

  - Overview tab: Personal + department quick stats
  - Analytics tab (NEW): Department trends & charts
  - Department tab: Faculty rankings
  - My PBAS tab: Personal forms
```

### Export Page
```
URL: http://localhost:3000/export
Features:
  - Export all sections (Dept, Faculty, Activity, Analytics, My PBAS)
  - Format selector (PDF/CSV)
  - Date range picker (defaults: 2020–current year)
  - Section toggles (8 sections)
```

## 📊 Feature Overview

### Three Main Systems

#### 1. Department Analytics
- **Access:** HOD Dashboard → Analytics tab
- **Features:** Charts, trends, faculty rankings
- **Export:** CSV download
- **Docs:** [HOD_ANALYTICS_USER_GUIDE.md](HOD_ANALYTICS_USER_GUIDE.md)

#### 2. PBAS PDF Reports
- **Access:** Export Page → My PBAS Data
- **Features:** VESIT-branded, comprehensive data
- **Export:** PDF format (professional)
- **Docs:** [PBAS_PDF_REPORT_ENHANCEMENT.md](PBAS_PDF_REPORT_ENHANCEMENT.md)

#### 3. CSV Exports
- **Access:** Export Page or Analytics tab
- **Features:** Spreadsheet-friendly, all details
- **Export:** CSV format (Excel-compatible)
- **Docs:** [PBAS_EXPORT_QUICK_REFERENCE.md](PBAS_EXPORT_QUICK_REFERENCE.md)
