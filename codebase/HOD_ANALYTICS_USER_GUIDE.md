# HOD Dashboard - Department Analytics Guide

## Quick Start

### Access Analytics Dashboard
1. Log in as HOD
2. Go to **HOD Dashboard**
3. Click the **Analytics** tab

## Features Overview

### 1. Filter Your Data

#### Year Range (2020-2026)
- Select **From Year:** Start of your analysis period
- Select **To Year:** End of your analysis period
- Use quick shortcuts for common ranges:
  - Last 3M, Last 6M, Last 1Y, Last 3Y

#### Categories
Choose which research activities to include:
- 📰 **Publications** - Books and conference proceedings
- 🔬 **Research** - Research papers and funded projects
- 🏆 **Patents** - Patent filings and intellectual property awards
- 📊 **Projects** - Research and consultancy projects
- 👥 **Guidance** - Student research supervision
- 📝 **Consultancy** - Consulting work and reviews

### 2. View Analytics

After setting filters, see:
- **Summary Cards** - Quick statistics at a glance:
  - Faculty count
  - Total publications/papers/patents
  - Projects and guidance instances
  
- **Yearly Trend Chart** - See how your department's output changed year by year
  
- **Category Distribution Pie Chart** - Understand which research areas dominate
  
- **Faculty-wise Breakdown Table** - Compare individual faculty contributions:
  - Name and email
  - Contribution count by category
  - Total output ranking

### 3. Export Data

#### Export as CSV
- Click **Export CSV** button
- Opens spreadsheet-friendly format
- Import to Excel or Google Sheets
- Includes summary section, yearly trends, and faculty details
- Filename: `Department_Analytics_[Dept]_[YearFrom]-[YearTo].csv`

#### Export as PDF
- From main dashboard, use the **Export Dept Data** button
- Select date range in dialog
- Choose CSV or PDF format
- Professional report with:
  - Department header
  - Summary statistics
  - Yearly trends table
  - Faculty breakdown table
  - Performance metrics

## Example Workflows

### Scenario 1: Annual Performance Report
1. Set Year: **2025 only** (From: 2025, To: 2025)
2. Select all categories
3. Click Export PDF
4. Share with administration or board

### Scenario 2: Research Impact Analysis
1. Set Year: **Last 3 Years** (2023-2025)
2. Select only: Research, Patents, Projects
3. View charts to understand research intensity
4. Export CSV for detailed analysis in Excel

### Scenario 3: Faculty Benchmarking
1. Set Year: **Full range** (2020-2026)
2. Select: Publications, Research Papers
3. Sort faculty table by total output
4. Identify top performers and support areas
5. Export CSV for individual feedback meetings

### Scenario 4: Consultancy Tracking
1. Set Year: **2026** (current year to date)
2. Select only: Consultancy category
3. Review faculty-wise consultancy breakdown
4. Export CSV for consultancy committee

## Data Interpretation Tips

### Understanding the Numbers
- **Publications:** Books, book chapters, conference proceedings
- **Research Papers:** Peer-reviewed journal articles
- **Patents:** Filed patents, trademarks, design registrations
- **Projects:** Funded research projects (leads and co-leads)
- **Guidance:** PhD/Masters student supervision
- **Consultancy:** Consulting assignments and reviews

### Reading the Charts
- **Yearly Trend:** Upward slope = increasing research output
- **Category Pie:** Larger slices = dominant research areas
- **Faculty Table:** Sorted by total (most productive first)

### Averages Explained
- **Avg Pub/Faculty:** Total publications ÷ faculty count
- **Avg Papers/Faculty:** Total papers ÷ faculty count
- **Avg Pub/Year:** Total publications ÷ years in range

## Troubleshooting

### No data showing?
- ✓ Check if faculty have submitted their PBAS forms
- ✓ Verify year range is correct
- ✓ Ensure at least one category is selected
- ✓ Click "Refresh" button to reload

### Export not working?
- ✓ Check your internet connection
- ✓ Ensure you have the CSV or PDF format selected
- ✓ Try exporting with fewer categories first
- ✓ Check browser console for errors

### Charts not displaying?
- ✓ Refresh the page
- ✓ Check if JavaScript is enabled
- ✓ Try a different browser
- ✓ Clear browser cache and try again
